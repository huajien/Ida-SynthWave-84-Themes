## 🔧 食用方式

1. **下载** GitHub存储库为.zip文件并解压缩。或者，你可以**克隆**存储库在您的终端使用以下命令:

   ```bash
   git clone https://github.com/huajien/Ida-SynthWave-84-Themes.git
   ```

2. **移动** SynthWave84 文件夹到IDA安装文件夹的' /themes '文件夹中。


## 👨‍🔧 修改主题

1. **打开** IDA并导航到 `Options -> Colors... -> Current theme`.
